# Global University Rankings Dashboard

## Project Overview

This Power BI dashboard explores global university rankings and performance metrics across countries and continents.

The report enables users to analyze university rankings, compare regions, and identify trends through interactive visualizations.

## Tools Used

- Power BI
- Power Query
- DAX

## Features

- University ranking analysis
- Top 10 universities visualization
- Country and continent comparisons
- Student distribution analysis
- Interactive filtering by year, continent and country
- Geographic visualization using maps
- KPI summary cards

## Dashboard Pages

### Executive Overview
Provides key metrics including:
- Best ranked university
- Median ranking
- Total universities
- Student distribution

### Ranking Analysis
Includes:
- Top ranked universities
- Average ranking by continent
- Top 100 university analysis

### Geographic Analysis
Provides:
- Country-level comparisons
- University distribution map
- Academic performance metrics

## Skills Demonstrated

- Data visualization
- Dashboard design
- KPI reporting
- DAX calculations
- Interactive reporting
- Business intelligence reporting
